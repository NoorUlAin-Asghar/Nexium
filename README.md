# 🚀 Nexium – AI Powered Web Applications

A collection of modern web applications built with **Next.js**, **TypeScript**, and **Tailwind CSS**, showcasing AI integrations, API consumption, authentication, database management, and responsive UI design.

---

## 🛠 Tech Stack

Across the repository, the projects utilize:

- Next.js 14+
- TypeScript
- Tailwind CSS
- ShadCN UI
- Framer Motion
- Supabase
- REST APIs
- Hugging Face API
- ApyHub API
- MyMemory Translation API

## 🚀 Getting Started

Clone the repository:

```bash
git clone https://github.com/NoorUlAin-Asghar/Nexium_NoorUlAinAsghar.git
```

Navigate to the project you want to run:

```bash
cd assignment-1
```

or

```bash
cd assignment-2
```

or

```bash
cd final-project
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Visit:

```
http://localhost:3000
```

---

## 🔑 Environment Variables

Some projects require API keys and Supabase credentials.

Create a `.env` file inside the corresponding project directory.

Depending on the project, you may need:

```env
NEXT_PUBLIC_SUPABASE_URL=
NEXT_PUBLIC_ANON_KEY=
API_KEY=
HF_TOKEN=
```

Refer to each project's README for complete setup instructions.
